# Nieuwe versie uitbrengen
Om het uitbrengen van een nieuwe versie zo makkelijk mogelijk te maken gaat dit via een DevOps build pipeline. Deze kan je hier terugvinden:
<a href="https://dev.azure.com/arlanet/Arlanet/_build?definitionId=139" target="_blank">https://dev.azure.com/arlanet/Arlanet/_build?definitionId=139</a>

## Versie ophogen
Het versienummer van de library is terug te vinden in het <code>Arlanet.Frontend.Library.csproj</code> bestand:
<img src={CsprojFile} />
Het gaat hier om de <code>PackageVersion</code> property. Als je deze vergeet op te hogen dan zal de build pipeline een error geven.

## Build pipeline draaien
Ga naar de <a href="https://dev.azure.com/arlanet/Arlanet/_build?definitionId=139" target="_blank">build pipeline</a> en klik rechtsboven op <code>Run pipeline</code>. 
Zodra dit proces klaar is kan de nieuwe versie gebruikt gaan worden in onze projecten. Zie daarvoor de installatie instructies.
