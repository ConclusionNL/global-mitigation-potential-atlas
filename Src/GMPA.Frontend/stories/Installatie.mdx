# Frontend library installeren

Het installeren van de frontend library gaat middels NuGet packages, de beschikbare versies zijn hier terug te vinden: 
[https://dev.azure.com/arlanet/Arlanet/_artifacts/feed/Frontend/NuGet/Arlanet.Frontend.Library/overview](https://dev.azure.com/arlanet/Arlanet/_artifacts/feed/Frontend/NuGet/Arlanet.Frontend.Library/overview)

## NuGet authenticatie
Helaas forceert het gebruik van DevOps artifacts het eenmalig moeten installeren van een NuGet plugin, de laatste versie kan <a href="https://github.com/microsoft/artifacts-credprovider/releases" target="_blank">hier</a> gevonden worden. 
Download <code>Microsoft.NuGet.CredentialProvider.zip</code> en plaats de inhoud van het plugins mapje in <code>%userprofile% > .nuget > plugins</code>

## Template installeren
Om de template van de frontend library op je computer te installeren, kan je het volgende _dotnet_ command gebruiken:

```
dotnet new install Arlanet.Frontend.Library --interactive --nuget-source https://pkgs.dev.azure.com/arlanet/Arlanet/_packaging/Frontend/nuget/v3/index.json
```
<div className="tip-wrapper mb-0">
  <span className="tip">Updaten naar nieuwe versie</span>Het bovenstaande command zorgt er ook voor dat de laatste versie wordt geinstalleerd indien er al een versie geinstalleerd is.
</div>
<div className="tip-wrapper mt-0">
  <span className="tip">Specifieke versie</span>Voeg <code>--version x.x.x</code> toe om een specifieke versie te installeren
</div>

Het kan zijn dat NuGet je gaat vragen om een Microsoft URL in je browser te openen en een code in te typen, als het goed is is dit eenmalig. Zodra het scherm aangeeft dat het gesloten kan worden kan je in je CLI op enter drukken om door te gaan.


## Project aanmaken/updaten
Het aanmaken van een project gaat via het volgende command:
```
dotnet new arlanetfrontendlibrary
```
<div className="tip-wrapper mb-0">
  <span className="tip tip-warning">Let op!</span>Dit zal plaatsvinden in de huidige directory! Maak vooraf dus een map aan voor je project
</div>
<div className="tip-wrapper mt-0">
  <span className="tip">Bestaande projecten</span>Voeg <code>--force</code> toe om bestaande projecten te updaten naar een nieuwe versie
</div>

## Troubleshooting
- Krijg je tijdens het installeren van de template de melding dat de package niet bestaat? <a href="https://dotnet.microsoft.com/en-us/download" target="_blank">Download de laatste .NET SDK</a> (versie >= 7.0.4)
